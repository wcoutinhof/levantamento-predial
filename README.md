# Levantamento Predial — PWA

Aplicativo web para levantamento de ambientes, tipologias de portas e janelas, quantitativos e memorial de cálculo.

## Publicar no GitHub Pages

1. Entre em sua conta do GitHub e crie um repositório, por exemplo: `levantamento-predial`.
2. Envie **todos os arquivos e pastas deste pacote** para a raiz do repositório.
3. Abra **Settings > Pages**.
4. Em **Build and deployment**, escolha **Deploy from a branch**.
5. Selecione a branch `main`, pasta `/ (root)` e clique em **Save**.
6. O endereço será semelhante a: `https://SEU-USUARIO.github.io/levantamento-predial/`.

## Instalar no iPhone

1. Abra o endereço publicado no Safari.
2. Toque em **Compartilhar**.
3. Escolha **Adicionar à Tela de Início**.
4. Abra pelo ícone criado.

## Uso e segurança dos dados

- Os dados são salvos localmente no navegador/aplicativo do aparelho.
- Use regularmente **Salvar projeto (.json)** para criar cópia de segurança.
- A opção **Abrir projeto** restaura o arquivo JSON em outro aparelho.
- A exportação CSV/Excel é destinada à consolidação e ao memorial de cálculo.
